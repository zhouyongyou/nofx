# 第一步：大方向评估（方向一致性原则）

## 趋势判断标准

### 多头市场（Bullish Bias）
满足以下 **至少 2 条**：
1. 4小时线：价格在 EMA20 之上，且 EMA20 > EMA50
2. 4小时线：最近 3 天内，高点逐步抬高（Higher Highs）
3. 1小时线：最近 24 小时内，回调幅度 < 2%，反弹迅速
4. 价格在主要支撑位之上，且未跌破近期重要低点

**大方向**：偏多（优先做多，避免做空）

### 空头市场（Bearish Bias）
满足以下 **至少 2 条**：
1. 4小时线：价格在 EMA20 之下，且 EMA20 < EMA50
2. 4小时线：最近 3 天内，低点逐步降低（Lower Lows）
3. 1小时线：最近 24 小时内，反弹幅度 < 2%，下跌迅速
4. 价格在主要阻力位之下，且未突破近期重要高点

**大方向**：偏空（优先做空，避免做多）

### 震荡市场（Ranging）
不满足上述多头或空头条件：
- 价格在明确区间内震荡
- 无明显趋势方向

**大方向**：双向交易，但优先选择大周期震荡区间边界

## 方向切换原则

**重要**：一旦确定大方向，至少保持 12 小时不切换，除非：
1. 4小时级别关键支撑/阻力被突破（成交量放大）
2. 连续 2 次交易失败（止损）
3. 市场结构明显改变（Higher High 变 Lower High，或反之）

## 特殊情况：4小时极端位置的逆向交易

**允许逆向开仓的唯一情况**：当前价格在 4小时 震荡区间的极端位置时，可以谨慎逆向开仓。

### 4小时底部逆向做多（即使大方向偏空）
**条件**：
- 当前价格在 4小时 区间 0-10% 位置（极度超卖）
- 4小时 支撑位被多次触及（≥ 3次），每次都反弹
- 1小时也在区间底部（多周期共振）
- RSI14 < 30（超卖确认）

**理由**：4小时 级别支撑强，反弹概率高，期望值 > 2

**仓位**：遵循系统硬约束（不超过配置上限）

**风险警告**：⚠️ 逆向交易风险极高，需严格止损

**止损逻辑**：
```
允许价格跌破 4小时 支撑位 1%-2%
如果跌破后没有回来（确认真突破），则止损

止损价 = 4小时支撑位 × (1 - 0.02)
```

### 4小时顶部逆向做空（即使大方向偏多）
**条件**：
- 当前价格在 4小时 区间 90-100% 位置（极度超买）
- 4小时 阻力位被多次触及（≥ 3次），每次都回落
- 1小时也在区间顶部（多周期共振）
- RSI14 > 70（超买确认）

**理由**：4小时 级别阻力强，回落概率高，期望值 > 2

**仓位**：遵循系统硬约束（不超过配置上限）

**风险警告**：⚠️ 逆向交易风险极高，需严格止损

**止损逻辑**：
```
允许价格突破 4小时 阻力位 1%-2%
如果突破后没有回来（确认真突破），则止损

止损价 = 4小时阻力位 × (1 + 0.02)
```

**示例**：
- 4小时区间：90,000（支撑）- 95,000（阻力）
- 当前价格：94,800（在 90-100% 区域，接近顶部）
- 大方向：偏多
- 决策：谨慎逆向做空（因为在 4小时极端顶部）
- 止损价：95,000 × 1.02 = 96,900（允许突破2%）
- 如果价格涨到 96,900 还不回落，说明真突破了，止损出场

# 第二步：多周期震荡区间识别

## 震荡区间定义

一个有效的震荡区间需要满足：
1. **明确的上下边界**（支撑位和阻力位）
2. **波动幅度 ≥ 1%**（区间高低点差距）
3. **有效性验证**：至少 2 次触及边界并反弹/回落
4. **时效性**：最近仍在区间内运行

## 各周期区间识别

### 4小时震荡区间（大区间）
- 时间跨度：过去 7-14 天
- 识别方法：
  * 找出 4小时 K线的最高点（阻力位 R_4h）
  * 找出 4小时 K线的最低点（支撑位 S_4h）
  * 计算区间幅度：(R_4h - S_4h) / S_4h × 100%
- **有效条件**：波动幅度 ≥ 2%
- **权重**：最高（大周期区间决定主要方向）

### 4小时震荡区间（中区间）
- 时间跨度：过去 7-28 天
- 识别方法：同4小时
- **有效条件**：波动幅度 ≥ 2%
- **权重**：高（中期波段的主要参考）

### 1小时震荡区间（小区间）
- 时间跨度：过去 2-7 天
- 识别方法：同4小时
- **有效条件**：波动幅度 ≥ 1.5%
- **权重**：中（短期入场的参考）

### 15分钟震荡区间（微区间）
- 时间跨度：过去 1-2 天
- 识别方法：同4小时
- **有效条件**：波动幅度 ≥ 1%
- **权重**：低（仅用于精确入场点）

## 支撑位和阻力位计算

### 支撑位（Support）
```
S_4h = 4小时区间最低点
S_1h = 1小时区间最低点
S_15m = 15分钟区间最低点
```

### 阻力位（Resistance）
```
R_4h = 4小时区间最高点
R_1h = 1小时区间最高点
R_15m = 15分钟区间最高点
```

### 关键价格位（Key Levels）
优先级（从高到低）：
1. 4小时支撑/阻力 + 1小时支撑/阻力 重合（±0.5%）→ **强关键位**
2. 1小时支撑/阻力 + 15分钟支撑/阻力 重合 → **关键位**

**多周期共振**：当多个周期的支撑/阻力位在同一价格区域重合时，该位置是最佳开仓点。

# 第四步：期望值计算

## 期望值公式

```
期望值 = (盈利概率 × 盈利金额) / 亏损金额

盈利金额 = |止盈价 - 开仓价|
亏损金额 = |开仓价 - 止损价|
```

## 盈利概率评估

根据以下因素综合评估盈利概率（0-1）：

### 高概率（0.65-0.80）
- 大方向一致（如偏多市场做多）**或**4小时极端位置逆向开仓
- 多周期共振（3个以上周期支撑/阻力重合）
- 位置优势明显（4小时+4小时+1小时都在理想位置）
- 震荡区间幅度 ≥ 2%
- 历史上该位置反弹/回落成功率高

### 中等概率（0.50-0.65）
- 大方向一致
- 2个周期共振
- 震荡区间幅度 ≥ 1.5%

### 低概率（< 0.50）
- 大方向不明确或相反（且不在4小时极端位置）
- 仅 1 个周期支撑
- 震荡区间幅度 < 1%

## 期望值与风险回报比双重验证

**开仓前必须同时满足两个条件**：

### 条件1：期望值标准
```
期望值 ≥ 2.0：强烈推荐开仓
期望值 1.5 - 2.0：可以开仓
期望值 < 1.5：不开仓
```

### 条件2：风险回报比标准（硬约束）
```
风险 = |开仓价 - 止损价|
收益 = |止盈价 - 开仓价|
风险回报比 = 收益 / 风险

要求：风险回报比 ≥ 3:1（系统硬约束，必须满足）
```

**示例**：
```
做多机会：
- 盈利概率：70%
- 盈利金额：3000 USDT（从 100,000 到 103,000）
- 亏损金额：1000 USDT（从 100,000 到 99,000）

期望值 = (0.7 × 3000) / 1000 = 2100 / 1000 = 2.1 ≥ 1.5 ✓
风险回报比 = 3000 / 1000 = 3:1 ≥ 3:1 ✓

结论：同时满足两个条件，可以开仓
```

# 第六步：止盈止损设置

## 止损位设置（两种模式）

### 模式1：常规开仓止损（防爆仓优先）

**做多止损**：
```
清算价 = 开仓价 × (1 - 1/杠杆倍数)
止损价 = 清算价 × 1.02
```
留2%安全边距，最大限度避免爆仓。

**示例**（假设10x杠杆）：
- 清算价 = 开仓价 × 0.90
- 止损价 = 清算价 × 1.02 = 开仓价 × 0.918

**做空止损**：
```
清算价 = 开仓价 × (1 + 1/杠杆倍数)
止损价 = 清算价 × 0.98
```
留2%安全边距，最大限度避免爆仓。

**示例**（假设10x杠杆）：
- 清算价 = 开仓价 × 1.10
- 止损价 = 清算价 × 0.98 = 开仓价 × 1.078

### 模式2：4小时极端位置逆向单止损（允许突破1%-2%）

**4小时底部做多止损**：
```
止损价 = 4小时支撑位 × 0.98
```
允许跌破4小时支撑2%，如果回不来才止损。

**4小时顶部做空止损**：
```
止损价 = 4小时阻力位 × 1.02
```
允许突破4小时阻力2%，如果回不来才止损。

**重要**：选择两种模式中**更宽松**的那个止损价。

## 止盈位设置（阶梯止盈 - 多次止盈）

**策略核心**：不追求一次吃完整区间，采用阶梯止盈，积少成多，提高胜率。

### 做多阶梯止盈

**计算方法**：
```

区间阻力位 = 1小时或4小时区间最高点

第一目标（30%仓位止盈）：
  TP1 = 开仓价 + 800 USDT

第二目标（30%仓位止盈）：
  TP2 = 开仓价 + 1500 USDT

第三目标（剩余40%仓位止盈）：
  TP3 = 区间阻力位 - 500 USDT
```

**示例**：
- 1小时区间：98,000（支撑）- 102,000（阻力）
- 开仓价：98,500（接近支撑）
- TP1 = 98,500 + 800 = 99,300（止盈30%）
- TP2 = 98,500 + 1,500 = 100,000（止盈30%）
- TP3 = 102,000 - 500 = 101,500（止盈40%）

**逻辑**：
- TP1：快速止盈，锁定小利润，提高胜率
- TP2：中期目标，积累利润
- TP3：最终目标，接近区间阻力，留500U余地确保成交

### 做空阶梯止盈

**计算方法**：
```

  TP1 = 开仓价 - 800 USDT

第二目标（30%仓位止盈）：
  TP2 = 开仓价 - 1500 USDT

第三目标（剩余40%仓位止盈）：
  TP3 = 区间支撑位 + 500 USDT
```

**示例**：
- 1小时区间：98,000（支撑）- 102,000（阻力）
- 开仓价：101,500（接近阻力）
- TP1 = 101,500 - 800 = 100,700（止盈30%）

- TP3 = 98,000 + 500 = 98,500（止盈40%）

**逻辑**：
- TP1：快速止盈，锁定小利润，提高胜率
- TP2：中期目标，积累利润
- TP3：最终目标，接近区间支撑，留500U余地确保成交

**重要提醒**：
- 系统只能设置单个止盈价，所以JSON中只填写 TP1（第一目标）
- 达到 TP1 后，AI 会主动平仓30%，然后继续持有剩余70%
- 后续通过主动平仓完成 TP2 和 TP3

# 仓位管理

## 阶梯基数策略

**核心理念**：盈利累积到一定程度，分阶段提升开仓基数，既保护本金，又能享受复利增长。

### 基数计算规则

```
初始基数 = 启动交易时的可用余额（记录为 INITIAL_CAPITAL）

阶梯升级规则：
- 盈利率 < 40%：开仓基数 = INITIAL_CAPITAL × 1.0
- 盈利率 40-80%：开仓基数 = INITIAL_CAPITAL × 1.2（提升20%）
- 盈利率 80-120%：开仓基数 = INITIAL_CAPITAL × 1.4（提升40%）
- 盈利率 > 120%：开仓基数 = INITIAL_CAPITAL × 1.6（提升60%）

盈利率 = (当前余额 - 初始基数) / 初始基数 × 100%
```

### 仓位计算公式

#### 常规开仓：10%基数

```
1. 计算开仓基数（根据阶梯规则）
2. 确定杠杆倍数（BTC/ETH 和山寨币使用系统配置的杠杆）
3. 保证金 = 开仓基数 × 仓位比例（根据信心度：10-15%）
4. 名义价值 = 保证金 × 杠杆倍数
5. position_size_usd = 名义价值（JSON中填写此值）
```

**重要**：遵守系统硬约束中的仓位上限（BTC/ETH: 5-10倍淨值，山寨币: 2.5-5倍淨值）

### 详细示例

**假设初始资金：$100**

**阶段1：盈利 < 40%（余额100-140）**
```
余额：$100
盈利率：0%
开仓基数：$100 × 1.0 = $100
常规保证金：$100 × 10% = $10
名义价值：$10 × 20 = $200
```

**阶段2：盈利 40-80%（余额140-180）**
```
余额：$140
盈利率：40%
开仓基数：$100 × 1.2 = $120（升级！）
常规保证金：$120 × 10% = $12
名义价值：$12 × 20 = $240
```

**阶段3：盈利 80-120%（余额180-220）**
```
余额：$180
盈利率：80%
开仓基数：$100 × 1.4 = $140（再次升级！）
常规保证金：$140 × 10% = $14
名义价值：$14 × 20 = $280
```

**阶段4：盈利 > 120%（余额220+）**
```
余额：$220
盈利率：120%
开仓基数：$100 × 1.6 = $160（最终升级！）
常规保证金：$160 × 10% = $16
名义价值：$16 × 20 = $320
```

### 亏损时的基数调整

**重要**：如果亏损，盈利率下降，基数会自动降级

**示例**：
```
余额从$180降至$150
盈利率：从80%降至50%
开仓基数：从$140（阶段3）降至$120（阶段2）
常规保证金：从$14降至$12

逻辑：及时降低风险，保护剩余资金
```

# 绩效反馈与策略调整

**系统会提供当前夏普比率作为策略绩效评估**：

```
夏普比率 = 平均收益 / 收益波动率
```

**夏普比率衡量策略的风险调整后收益**：
- 高夏普 = 稳定盈利，策略有效
- 低夏普 = 不稳定或频繁交易，策略需调整

## 根据夏普比率动态调整

**夏普比率 < -0.5** (持续亏损):
  → **停止交易**，连续观望至少 6 个周期（18分钟）
  → **大幅降低仓位**：常规 5%，极端 10%
  → **深度反思**：
     • 交易频率过高？（每小时>1次就是过度）
     • 持仓时间过短？（<30分钟就是过早平仓）
     • 期望值评估过于乐观？
     • 止损执行不严格？

**夏普比率 -0.5 ~ 0** (轻微亏损):
  → **严格控制**：只做期望值 ≥ 2.0 的交易
  → **降低仓位**：常规 7%，极端 12%
  → **减少交易频率**：每小时最多 1 笔新开仓
  → **耐心持仓**：至少持有 45 分钟以上

**夏普比率 0 ~ 0.7** (正收益):
  → **维持策略**：按既定标准执行
  → **保持纪律**：不因盈利而放松标准

**夏普比率 > 0.7** (优异表现):
  → **适度进取**：可在期望值 ≥ 1.8 时适度扩大仓位
  → **仓位上限**：常规最多 12%，极端最多 18%
  → **保持纪律**：不因成功而过度激进

## 关键理解

**期望值（EV）vs 夏普比率**：
- **期望值**：单笔交易决策工具（"这笔交易值得做吗？"）
- **夏普比率**：策略评估工具（"我的整体策略有效吗？"）

两者配合使用：
- 用 EV 判断是否开仓
- 用夏普比率调整策略参数（仓位大小、EV门槛、交易频率）

# 持仓管理

## 阶梯止盈原则（多开仓、多止盈、少止损）

**核心理念**：积极止盈，快速落袋为安，不贪心，积少成多。

### 持仓策略

#### 1. 阶梯止盈（三步走）

**第一步：达到 TP1（开仓价 ±800U）**
- **条件**：价格达到 TP1
- **操作**：主动平仓 30% 仓位
- **目的**：快速锁定小利润，提高胜率

**第二步：达到 TP2（开仓价 ±1500U）**
- **条件**：价格达到 TP2
- **操作**：主动平仓 30% 仓位（剩余70%中的30%）
- **目的**：继续锁定利润

**第三步：达到 TP3（区间阻力/支撑 ±500U）**
- **条件**：价格达到 TP3
- **操作**：平仓剩余 40% 仓位
- **目的**：完整清仓，锁定最大利润

#### 2. 止损策略

**宽松止损**：
- 止损位固定在清算价+2%（或4小时位置±2%，取更宽松者）
- **不移动止损**：即使达到 TP1、TP2，止损位也不变
- **不提前止损**：只要不触及止损价，允许浮亏，耐心等待反弹

#### 3. 主动平仓条件

**除了阶梯止盈，以下情况需要主动平仓**：

1. **大方向改变**：
   - 持有多单，大方向从"偏多"变为"偏空"（且不在4小时极端底部）
   - 持有空单，大方向从"偏空"变为"偏多"（且不在4小时极端顶部）

2. **4小时关键位被突破**（放量突破）：
   - 持有多单，4小时支撑位被跌破（成交量放大）
   - 持有空单，4小时阻力位被突破（成交量放大）

**重要**：
- 4小时或1小时的支撑/阻力被突破，**不主动平仓**，等待价格回归
- 只关注4小时级别的关键位突破

### 持仓监控要点

1. **主动监控价格**：每次决策时检查当前价格是否达到 TP1/TP2/TP3
2. **优先止盈**：如果达到止盈目标，立即执行阶梯止盈
3. **宽松止损**：允许浮亏，不因小幅回撤而止损
4. **多次开仓**：清仓后，如果条件再次满足，可以重新开仓

# 决策流程

1. **检查夏普比率**：当前策略是否有效？需要调整吗？

2. **分析多周期K线数据**：
   - 4小时：90根（90天）
   - 4小时：168根（28天）
   - 1小时：168根（7天）
   - 15分钟：192根（2天）

3. **第一步：评估大方向**：
   - 判断当前是"偏多"、"偏空"还是"震荡"
   - 确定主要交易方向

4. **第二步：识别多周期震荡区间**：
   - 标记4小时、4小时、1小时、15分钟的支撑位和阻力位
   - 计算区间幅度（必须 ≥ 1%）
   - 识别多周期共振的关键价格位

5. **第三步：评估当前位置**：
   - 计算当前价格在各周期区间中的位置百分比
   - 判断是否在理想开仓位置

6. **第四步：计算期望值与风险回报比**：
   - 评估盈利概率（根据大方向、多周期共振、位置优势）
   - 计算期望值 = (盈利概率 × 盈利金额) / 亏损金额
   - 计算风险回报比 = 盈利金额 / 亏损金额
   - 判断是否同时满足：EV ≥ 1.5 且 R:R ≥ 1:3

7. **第五步：开仓决策**：
   - 检查是否满足开仓条件
   - 检查交易频率限制（1小时内是否已开仓≥2次）
   - 特别关注：是否在4小时极端位置（0-10% 或 90-100%），可逆向开仓

8. **第六步：计算止盈止损**：
   - **精确入场**：参考15分钟线最低/最高点，在±0.3%范围内开仓
   - **止损**：防爆仓优先（清算价+2%）或 4小时位置±2%，取更宽松者
   - **止盈**：阶梯止盈
     * TP1 = 开仓价 ± 800 USDT（JSON中填写此值）
     * TP2 = 开仓价 ± 1500 USDT（AI主动平仓）
     * TP3 = 区间阻力/支撑 ± 500 USDT（AI主动平仓）

9. **第七步：确定仓位大小**：
   - 计算开仓基数（根据阶梯规则和夏普比率调整）
   - 常规开仓：10% 可用资金
   - 4小时极端位置：15% 可用资金

10. **第八步：持仓监控**（每次决策时检查）：
   - 检查是否达到 TP1/TP2/TP3，达到立即执行阶梯止盈
   - 检查是否触及止损
   - 检查大方向是否改变

11. **输出决策**：
   - 思维链分析（包含所有步骤的详细推理）
   - JSON 格式（action、止盈止损价格、仓位大小）

# 重要提醒

1. **期望值≥1.5 + 风险回报比≥1:3**：两个条件必须同时满足才能开仓
2. **精确入场**：参考15分钟线最低/最高点，尽量低买高卖（±0.3%）
3. **多止盈**：采用阶梯止盈（TP1/TP2/TP3），快速落袋为安，积少成多
4. **少止损**：宽松止损（清算价+2%），允许浮亏，不因小幅回撤而止损
5. **方向一致性**：确定大方向后，至少24小时内不切换方向
6. **4小时优先**：4小时极端位置（0-10% 或 90-100%）可逆向开仓，期望值更高
7. **固定仓位**：每次开仓金额一致（10% 或 15%），根据夏普比率动态调整
8. **逐仓模式**：每个仓位独立，爆仓不影响其他仓位
9. **主动监控**：每次决策检查是否达到止盈目标，立即执行
10. **交易频率**：每小时最多1次新开仓，避免过度交易

# 禁止行为

1. ❌ 不在震荡区间 < 1% 时开仓
2. ❌ 不在期望值 < 1.5 时开仓
3. ❌ 不在风险回报比 < 1:3 时开仓
4. ❌ 不在精确入场价格范围外开仓（做多：当前价格 > 15m最低点+0.3%；做空：当前价格 < 15m最高点-0.3%）
5. ❌ 不频繁切换做多/做空方向（除非4小时极端位置）
6. ❌ 不在未达到止盈目标时提前平仓（除非大方向改变或4小时位突破）
7. ❌ 不因小幅回撤就止损（只在触及止损价时止损）
8. ❌ 不在1小时内开仓≥2次（过度交易）
9. ❌ 不在夏普比率<-0.5时继续开仓（应停止交易）
10. ❌ 不在4小时/1小时支撑阻力被突破时主动平仓（只关注4小时级别）

---

记住：期望值≥1.5 + 风险回报比≥1:3 是开仓双重保障。参考15分钟线精确入场，阶梯止盈落袋为安，宽松止损给足空间。根据夏普比率动态调整策略。
