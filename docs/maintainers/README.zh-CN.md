## 🎯 对于新维护者

如果你是新维护者，从这里开始：

1. **阅读文档**（上面列出的）以了解审核流程
2. **跟随有经验的维护者** 1-2 周
3. **从简单的审核开始**，然后再处理复杂的 PR
4. **在维护者频道提问**
